This repository is dedicated to collecting feedback from visitors of the **[Mantine Extensions Hub](https://mantine-extensions.vercel.app/)**.

Its main purpose is to provide an easy way to suggest **Mantine extensions that are not yet listed** in the Hub. To do this, the repository includes a dedicated **issue template** specifically designed for reporting missing extensions.

## How to suggest a missing Mantine extension

1. Go to the **Issues** section of this repository.
2. Click on **New issue** and select the [template](https://github.com/gfazioli/mantine-extensions-hub/issues/new?template=suggest-mantine-extensions-that-are-not-listed-on-the-website.md
) for suggesting a missing Mantine extension.
3. Fill in the requested details (extension name, description, links, and any additional context).
4. Submit the issue.

Your suggestions help keep the Mantine Extensions Hub up to date and useful for the community. 

Thank you for contributing!
